# Исправление ошибки WebView2Loader.dll на Windows

Если при запуске `CarShareX-1.0.0-windows.exe` вы видите ошибку:
```
Не удается продолжить выполнение кода, поскольку система не обнаружила WebView2Loader.dll
```

## Быстрое решение

### Способ 1: Прямая ссылка для скачивания (рекомендуется)

Скачайте и установите Microsoft Edge WebView2 Runtime:

**Прямая ссылка на установщик:**
https://go.microsoft.com/fwlink/p/?LinkId=2124703

### Способ 2: С официального сайта

1. Откройте в браузере: https://developer.microsoft.com/microsoft-edge/webview2/
2. Прокрутите до раздела "Download the Runtime"
3. Выберите один из вариантов:
   - **Evergreen Bootstrapper** (маленький файл ~2MB, скачивает остальное онлайн)
   - **Evergreen Standalone Installer** (большой файл ~150MB, автономная установка)

4. Запустите скачанный файл
5. Дождитесь завершения установки
6. Запустите CarShareX заново

## Что такое WebView2?

WebView2 - это компонент от Microsoft, который позволяет приложениям отображать веб-контент. Он используется многими современными приложениями и уже установлен на большинстве компьютеров с Windows 10/11.

После установки WebView2 Runtime один раз, он будет работать для всех приложений на базе Tauri.

## Проверка установки

После установки WebView2:
1. Закройте все окна CarShareX (если они открыты)
2. Запустите `CarShareX-1.0.0-windows.exe` заново
3. Приложение должно открыться без ошибок

## Если проблема сохраняется

Если после установки WebView2 проблема не решилась:

1. **Перезагрузите компьютер**
2. Попробуйте запустить приложение от имени администратора (правый клик → "Запуск от имени администратора")
3. Проверьте, что у вас установлена последняя версия Windows 10 или Windows 11

## Системные требования

- Windows 10 версия 1809 или новее
- Windows 11 (любая версия)
- 200 MB свободного места на диске
- Подключение к интернету для скачивания WebView2 (только один раз)

---

**Примечание**: WebView2 полностью бесплатен и распространяется Microsoft.
