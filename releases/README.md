# CarShareX Desktop - Релизы

Готовые к использованию десктоп-приложения CarShareX для различных операционных систем.

## Версия 1.0.0

### Для Linux (RedOS, Ubuntu, Fedora, и др.)

#### DEB пакет (Debian/Ubuntu/RedOS)
- **Файл**: `CarShareX_1.0.0_amd64.deb`
- **Размер**: 3.1 MB
- **Установка**:
  ```bash
  sudo dpkg -i CarShareX_1.0.0_amd64.deb
  sudo apt-get install -f  # Установка зависимостей, если требуется
  ```
- **Запуск**: Найдите "CarShareX" в меню приложений или запустите `carsharex` в терминале

#### RPM пакет (RedOS/Fedora/RHEL)
- **Файл**: `CarShareX-1.0.0-1.x86_64.rpm`
- **Размер**: 3.1 MB
- **Установка**:
  ```bash
  sudo rpm -i CarShareX-1.0.0-1.x86_64.rpm
  # или для RedOS/Fedora:
  sudo dnf install ./CarShareX-1.0.0-1.x86_64.rpm
  ```
- **Запуск**: Найдите "CarShareX" в меню приложений или запустите `carsharex` в терминале

### Для Windows

⚠️ **ВАЖНО**: Для работы приложения требуется Microsoft Edge WebView2 Runtime

#### Файлы для скачивания
- **Приложение**: `CarShareX-1.0.0-windows.exe` (19 MB)
- **Установщик WebView2**: `install-webview2.bat` (автоматический установщик)
- **Исправление DLL ошибки**: `fix-dll-windows.bat` (если видите ошибку WebView2Loader.dll)

#### Установка (ВАЖНО: следуйте по порядку)

**Способ 1: Автоматический установщик (САМЫЙ ПРОСТОЙ)**

1. **Запустите** `install-webview2.bat`
2. Нажмите любую клавишу для продолжения
3. Дождитесь завершения установки WebView2
4. **ПЕРЕЗАГРУЗИТЕ КОМПЬЮТЕР** (это важно!)
5. **Теперь запустите** `CarShareX-1.0.0-windows.exe`
6. Пройдите предупреждение SmartScreen: "Дополнительно" → "Выполнить в любом случае"

**Если после этого видите ошибку "WebView2Loader.dll":**
1. Запустите `fix-dll-windows.bat` в той же папке
2. Попробуйте запустить CarShareX снова

**Способ 2: Ручная установка WebView2**

1. **Скачайте** WebView2 Runtime по прямой ссылке:
   ```
   https://go.microsoft.com/fwlink/p/?LinkId=2124703
   ```
2. **Запустите** скачанный файл `MicrosoftEdgeWebview2Setup.exe`
3. **Дождитесь** завершения установки
4. **ПЕРЕЗАГРУЗИТЕ КОМПЬЮТЕР** (это важно!)
5. **Теперь запустите** `CarShareX-1.0.0-windows.exe`
6. Пройдите предупреждение SmartScreen: "Дополнительно" → "Выполнить в любом случае"

**Если после этого видите ошибку "WebView2Loader.dll":**
1. Запустите `fix-dll-windows.bat` в той же папке
2. Или вручную скопируйте `WebView2Loader.dll` из `C:\Program Files (x86)\Microsoft\EdgeWebView\Application\[версия]\` в папку с CarShareX

**Способ 3: С официального сайта Microsoft**

1. Откройте https://developer.microsoft.com/microsoft-edge/webview2/
2. Скачайте "Evergreen Bootstrapper" или "Evergreen Standalone Installer"
3. Установите скачанный файл
4. Запустите CarShareX

**Почему это нужно?**

WebView2 Runtime - это бесплатный компонент от Microsoft, который используется для отображения веб-контента в приложениях. Он устанавливается **один раз** и используется всеми Tauri приложениями на вашем компьютере.

## Системные требования

### Linux
- RedOS 7.3+, Ubuntu 20.04+, Fedora 35+, или совместимый дистрибутив
- GTK 3.22+
- WebKit2GTK 2.28+
- Минимум 100 MB свободного места

### Windows
- Windows 10 версия 1809+ или Windows 11
- WebView2 Runtime (устанавливается автоматически при первом запуске)
- Минимум 200 MB свободного места

## Особенности

- Нативная производительность
- Малый размер (использует системный WebView)
- Все функции веб-версии
- Автономная работа (после загрузки данных)
