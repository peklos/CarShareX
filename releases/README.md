# CarShareX Desktop - Релизы

Готовые к использованию десктоп-приложения CarShareX для различных операционных систем.

## Версия 1.0.0

### Для Linux (RedOS, Ubuntu, Fedora, и др.)

#### DEB пакет (Debian/Ubuntu/RedOS)
- **Файл**: `CarShareX_1.0.0_amd64.deb`
- **Размер**: 3.1 MB
- **Установка**:
  ```bash
  sudo dpkg -i CarShareX_1.0.0_amd64.deb
  sudo apt-get install -f  # Установка зависимостей, если требуется
  ```
- **Запуск**: Найдите "CarShareX" в меню приложений или запустите `carsharex` в терминале

#### RPM пакет (RedOS/Fedora/RHEL)
- **Файл**: `CarShareX-1.0.0-1.x86_64.rpm`
- **Размер**: 3.1 MB
- **Установка**:
  ```bash
  sudo rpm -i CarShareX-1.0.0-1.x86_64.rpm
  # или для RedOS/Fedora:
  sudo dnf install ./CarShareX-1.0.0-1.x86_64.rpm
  ```
- **Запуск**: Найдите "CarShareX" в меню приложений или запустите `carsharex` в терминале

### Для Windows

- **Файл**: `CarShareX-1.0.0-windows.exe`
- **Размер**: 19 MB

#### Установка

**Вариант 1: Автоматическая установка (рекомендуется)**
1. Запустите `CarShareX-1.0.0-windows.exe`
2. Пройдите предупреждение SmartScreen: "Дополнительно" → "Выполнить в любом случае"
3. Приложение автоматически установит WebView2 Runtime при первом запуске (требуется интернет)

**Вариант 2: Если автоматическая установка не работает**

Если при запуске вы видите ошибку `WebView2Loader.dll`:

1. **Скачайте WebView2 Runtime вручную**:
   - Перейдите на https://developer.microsoft.com/microsoft-edge/webview2/
   - Или прямая ссылка: https://go.microsoft.com/fwlink/p/?LinkId=2124703
   - Скачайте "Evergreen Bootstrapper" или "Evergreen Standalone Installer"

2. **Установите WebView2**:
   - Запустите скачанный установщик
   - Дождитесь завершения установки

3. **Запустите CarShareX снова**:
   - Теперь приложение должно запуститься нормально

**Примечание**: WebView2 Runtime устанавливается один раз и используется всеми приложениями на базе Tauri

## Системные требования

### Linux
- RedOS 7.3+, Ubuntu 20.04+, Fedora 35+, или совместимый дистрибутив
- GTK 3.22+
- WebKit2GTK 2.28+
- Минимум 100 MB свободного места

### Windows
- Windows 10 версия 1809+ или Windows 11
- WebView2 Runtime (устанавливается автоматически при первом запуске)
- Минимум 200 MB свободного места

## Особенности

- Нативная производительность
- Малый размер (использует системный WebView)
- Все функции веб-версии
- Автономная работа (после загрузки данных)
